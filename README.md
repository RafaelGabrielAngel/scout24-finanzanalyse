## Inhalt

Interaktives Equity-Research-Dashboard zur Scout24 SE mit folgenden Modulen:

| Tab | Inhalt |
|---|---|
| Executive Summary | Investment-These, Football Field, KPI-Übersicht |
| DCF Live-Modell | Interaktives DCF mit Echtzeit-Slidern & Sensitivitätsmatrix |
| Peer Multiples | Peer-Vergleich & Bewertungsmatrix |
| Übernahme-Szenario | KKR/Immowelt, Spanien, M&A-Prämien, Goodwill-Risiko |
| Geschäftsmodell-Lücken | 5 strukturelle Short-Thesen, Revenue-at-Risk, Disruptions-Timeline |
| Investment-Urteil | Risiken, Catalyst Calendar, SWOT |

## Technische Umsetzung

- Reines HTML/CSS/JavaScript — keine Build-Tools, keine Dependencies
- Chart.js 4.4.1 (CDN)
- Vollständig offline-fähig (außer Chart.js-CDN)
- Responsive für Desktop & Mobile

## Nutzung

```bash
# Einfach im Browser öffnen:
open index.html
```

Oder via GitHub Pages / Netlify als statische Website deployen.
